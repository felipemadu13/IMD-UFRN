PERGUNTA:
Entrada e saída!
O uso das instruções primitivas é essencial no desenvolvimento de algoritmos. Raras serão as situações que permitirão construir algoritmos que não utilizem dados de entrada ou não possuam saída de dados.

Sua primeira missão, como programador, será construir um pequeno programa que irá receber 3 dados distintos:

Um valor X do tipo texto
Um valor Y do tipo inteiro
Um valor Z do tipo número
Esses 3 dados serão, respectivamente, um nome, uma idade e uma altura em metros. Após o seu programa receber essas informações o seu programa deverá escrever na tela no seguinte formato:

Eu sou X
Tenho Y anos
Minha altura é de Zm


RESPOSTA:
var nome := leia_texto
var idade := leia_inteiro
var altura := leia_numero

escreva "Eu sou {nome}"
escreva "Tenho {idade} anos"
escreva "Minha altura é de {altura}m"
