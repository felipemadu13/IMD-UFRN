Uma chave estrangeira é um identificador único de uma relação entre tabelas. Sua função é ser um atributo que faz a relação entre uma tabela B com a chave primária de uma tabela A. Uma chave estrangeira faz com que os dados entre duas tabelas sejam consistentes. Um exemplo poderia ser as tabelas Aluno e Projeto, onde a matrícula é a chave primária da tabela Aluno e na tabela Projeto teríamos uma chave estrangeira para identificar a qual matrícula aquele projeto pertence.
