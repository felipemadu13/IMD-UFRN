Quais as características principais que diferem os bancos NoSQL dos Relacionais?

NoSQL (not only SQL) é um termo para designar banco de dados não relacionais,
que são utilizados para se trabalhar com dados não estruturados.

Entre as principais diferenças entre os bancos de dados NoSQL e relacionais está:

Estrutura: Os dados armazenados em banco de dados relacionais são armazenados em tabelas com linhas e colunas, enquanto os bancos de dados NoSQL podem ser armazenados em documentos.

Linguagem: Os bancos NoSQL possuem uma linguagem de consulta própria e podem oferecer suporte a linguagem SQL.

Escalabilidade: O custo de escalabilidade em bancos de dados NoSQL são mais baixos se comparados aos bancos relacionais.
